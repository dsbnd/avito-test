# Avito Tech Internship 2026 — Дарья Бондаренко

Репозиторий содержит выполнение тестового задания на позицию стажера-тестировщика (QA Automation).

## Структура проекта

* **[Task 1](./task-1/README.md)** — Анализ функциональности, создание тест-кейсов и чек-листа для ручного тестирования.
* **[Task 2](./task-2/README.md)** — Автоматизация критических сценариев (UI-тесты) с использованием Java и Selenium.

## Как работать с репозиторием
Каждая папка автономна. Для ознакомления с результатами ручного тестирования перейдите в директорию `task-1`. Инструкции по запуску автотестов находятся внутри директории `task-2`.
